# Questions
## Which converges faster?
The second, ${\pi \over 6} = \tan^{-1}{1 \over \sqrt{3}}$

## Why?
Because ${1 \over \sqrt{3}} < 1$, for all integer $n > 0$ , ${{1 \over \sqrt{3}}^n \over n}$ converge faster than $1^n \over n$
